
# フィギュア定義書 (Figures Definition)

`scripts/analyze_data.py` (学習データ分析スクリプト) によって生成された可視化グラフの一覧と定義です。

## 1. データ分布 (Data Distribution)

### `finish_pos_dist.png` (着順分布)
- **定義**: 学習データの「着順（1着〜18着）」ごとの頻度分布。
- **目的**: 
    - データの順位に異常値（例: 存在しない順位）が含まれていないかの確認。
    - 出走頭数の傾向把握（右側のバーが低いのは、少頭数のレースが一定数あるため）。

### `missing_values.png` (欠損率の可視化)
- **定義**: 各特徴量における「欠損値（NaN）」の割合（%）を棒グラフで並べたもの。
- **目的**: 
    - データの欠損状況を把握し、入力精度の確認（バグ検出）に利用。
    - 欠損値補完（Imputation）が必要な項目の特定。
    - **重要**: 今回の分析で「条件近似特徴量」が100%欠損していることが判明したきっかけとなったグラフです。

## 2. 特徴量の分布 (Feature Distributions)

### `dist_*.png` (各特徴量のヒストグラム)
個別の数値特徴量の頻度分布を示します。形状（正規分布に近いか、歪んでいるか）を確認します。

- **`dist_speed_best_5.png`**:
    - **過去5走の最高スピード指数**。指数（Speed Index）が高いほど速いです。
- **`dist_closing_best_5.png`**:
    - **過去5走の最高上がり3ハロン指数**。指数が高いほど終いの脚が鋭いです。
- **`dist_early_mean_3.png`**:
    - **過去3走の平均先行指数**（Early Index）。負の値を取り、絶対値が大きい（より小さい負の値）ほど前の位置取りを意味します。
- **`dist_speed_best_5_z_inrace.png`**:
    - **レース内相対スピード指数 (Z-score)**。そのレースのメンバー内での相対的な速さを表します（平均0）。
- **`dist_body_weight.png`**:
    - **馬体重**。400kg〜550kg程度に分布します。

### `outliers_boxplot.png` (外れ値検出箱ひげ図)
- **定義**: 主要な数値特徴量を標準化（平均0, 分散1）し、箱ひげ図で並べたもの。
- **目的**: 
    - 特徴量ごとのばらつきや外れ値（極端な値）を横並びで比較・確認するため。
    - モデル学習において悪影響を与える極端な値がないかチェックします。

## 3. 相関関係 (Correlations)

### `correlation_matrix.png` (相関行列ヒートマップ)
- **定義**: 各数値特徴量間の「相関係数（-1.0 〜 +1.0）」をヒートマップで可視化したもの。
    - **赤色**: 正の相関（片方が増えれば、もう片方も増える）
    - **青色**: 負の相関（片方が増えれば、もう片方は減る）
- **目的**: 
    - **目的変数（`is_win`）との関係**: 勝ちに直結する重要な特徴量を見つける。
    - **多重共線性（Multicollinearity）の確認**: 説明変数同士で似すぎている項目がないか（モデルの不安定要因になるため）を確認する。
